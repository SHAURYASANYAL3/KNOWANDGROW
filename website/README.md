# KNOW And GROW Website Scaffold

This folder contains an initial static website scaffold for the KNOW And GROW project.

## Included pages
- `index.html` — Homepage with mission, CTA, and focus highlights
- `about.html` — Organization mission, approach, and trust points
- `focus-areas.html` — Coverage of the five core youth topics
- `resources.html` — Resource listings and learning content
- `emergency-help.html` — Emergency guidance and safe support instructions
- `volunteer.html` — Volunteer opportunities and sign-up form
- `contact.html` — Contact form and contact details
- `privacy.html` — Privacy and data protection statement

## Assets
- `styles.css` — Shared layout, responsive styling, and accessible visual design
- `script.js` — Mobile menu toggle for small screens

## Notes
- Links are intentionally placeholder or dummy where appropriate.
- This scaffold is designed as a planning prototype for the NGO website.
- You can extend it into a full Next.js/Tailwind site later if needed.
